## 🚀 方案一：Firebase Hosting 部署（推荐）

### 第一步：打开命令行

1. 按 `Win + R`，输入 `cmd`
2. 进入项目目录：
```bash

```

### 第二步：登录 Firebase

```bash
npx firebase login
```

会打开浏览器，选择您的 Google 账号登录。

### 第三步：初始化项目

```bash
npx firebase init hosting
```

选择：
- 项目：`tool-77737`（使用现有项目）
- 公共目录：`www`
- 单页应用：`Yes`（所有请求重定向到 index.html）
- GitHub Actions：`No`

### 第四步：部署

```bash
npx firebase deploy
```

### 第五步：获取网址

部署成功后会显示：
```
Hosting URL: https://tool-77737.web.app
```

**这就是您的公网访问地址！手机和电脑都能访问！**

## 🔄 双向实时同步原理

您的应用已经实现了实时同步：

1. **电脑修改数据** → 自动上传到 Firebase
2. **Firebase 监听变化** → 推送到所有连接的设备
3. **手机收到更新** → 页面自动刷新

代码中的关键实现：
```javascript
// 监听云端变化
onSnapshot(doc(db, 'users', uid), (doc) => {
  if (doc.exists()) {
    // 自动同步到本地
    mergeCloudData(doc.data());
  }
});
```

## 🆘 遇到问题？

### Q: Firebase 部署失败？
尝试清除缓存后重新部署：
```bash
npx firebase logout
npx firebase login
npx firebase deploy
```

### Q: 手机无法访问？
确保：
1. 手机和电脑网络正常
2. 使用的是公网地址（不是 localhost）
3. Firebase 控制台中已添加域名授权

### Q: 同步不生效？
检查 Firebase 控制台：
1. Firestore Database 中是否有数据
2. Authentication 中是否有用户记录
